## 🛠️ 快速恢复工作指南 (Checklist for New Machine)
如果你在另一台电脑上拉取了代码，请按以下步骤开始工作：

1. **获取代码**:
   ```bash
   git pull origin main
   ```

2. **恢复依赖**:
   ```bash
   cd AutoKeep
   npm install
   ```

3. **配置环境变量**:
   *注意：`.env` 文件通常不会上传到 GitHub，你需要手动创建 `AutoKeep/.env` 文件并填入以下信息：*
   ```
   VITE_SUPABASE_URL=你的_Supabase_URL
   VITE_SUPABASE_ANON_KEY=你的_Supabase_Anon_Key
   ```

4. **启动开发服务器**:
   ```bash
   npm run dev
   ```

5. **构建并发布 (可选)**:
   ```bash
   npm run deploy
   ```
